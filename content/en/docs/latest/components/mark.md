## Overview

{{< release version="v0.19.0" >}}

Use the `mark` shortcode to highlight text. The inner content is used as input.

<!-- markdownlint-disable MD037 -->
{{< example lang="hugo" >}}
Use the mark shortcode to {{</* mark >}}highlight{{< /mark */>}} specific text.
{{< /example >}}
